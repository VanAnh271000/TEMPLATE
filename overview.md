# Project Overview

This project is an enterprise-ready backend template built with ASP.NET Core.

## Goals
- Clean Architecture
- Scalable & maintainable codebase
- Production-ready monitoring and logging
- Asynchronous background processing
- Performance and caching best practices

## Tech Stack
- ASP.NET Core
- Entity Framework Core
- Serilog + Seq
- OpenTelemetry
- Prometheus + Grafana
- Redis
- Hangfire
- Docker

## Key Features
- RESTful API with versioning
- Structured logging and correlation ID
- Metrics and performance monitoring
- Notification system (Email, SMS, Firebase)
- Distributed caching with Redis
